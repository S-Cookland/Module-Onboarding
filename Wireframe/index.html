What is a branch in Git?

A branch represents an independent line of development. Branches serve as an abstraction for the edit/stage/commit process. You can think of them as a way to request a brand new working directory, staging area, and project history.

What is the purpose of README?

The README file acts like the project's own homepage and introduces the project to the user. Essential information is placed here to allow users and developers to understand, use, and contribute to the project. It is the entry point to the project's documentation.

What is the purpose of a wireframe?

Acting as a blueprint for a digital product, such as a website or mobile app allowing users to visualise and plan the layout, structure, and interface elements. Focused on functionality and user flow, helping teams to align, test, and refine concepts quickly.
